## Stripe:

---

### **_Instalar Stripe-CLI:_**

Instalar de acordo com o SO: https://stripe.com/docs/stripe-cli

### **Rodar Stripe Interceptopr Webhook localmente:**

```sh
stripe listen --forward-to localhost:3000/api/webhooks
```
